### Краткое руководство

Учебный проект для автоматического сбора 
данных о книгах с сайта Books to Scrape
(https://books.toscrape.com). 
Реализованы функции:
* парсинг всех страниц сайта, 
* извлечение информации о книгах, 
* автоматический ежедневный запуск задачи,
* опциональное сохранение результата. 

#### Запуск приложения:
python scrape_books.py

#### Запуск тестов
python -m pytest tests/

#### Cписок используемых библиотек.
* beautifulsoup4==4.14.2
* pytest==8.4.2
* requests==2.32.5
* schedule==1.2.2
